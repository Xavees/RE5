# Resident Evil 5 - Tradução PT-BR (Central de Tradução)

## Sobre

Este repositório reúne os arquivos da tradução em português brasileiro de **Resident Evil 5**, desenvolvida originalmente pela **Central de Tradução**.

Todo o crédito pelo trabalho de tradução pertence à equipe da Central de Tradução. Este repositório existe apenas para facilitar o acesso e a organização dos arquivos.

## Créditos

* Tradução: Central de Tradução
* Jogo: Capcom

Agradecimentos à equipe da Central de Tradução por disponibilizar este excelente trabalho para a comunidade brasileira.

## Recomendação

✅ **Versão recomendada:** Tradução com legendas.

A versão legendada oferece a melhor experiência, preservando a qualidade original das vozes e garantindo uma tradução consistente durante toda a campanha.

⚠️ **Versão dublada por IA**

A dublagem disponível foi produzida utilizando Inteligência Artificial e possui caráter experimental. A qualidade das vozes pode variar, apresentando problemas de entonação, sincronização e interpretação em algumas cenas. Caso esteja buscando a melhor experiência possível, recomenda-se utilizar apenas a versão legendada.

## Aviso

Este repositório não reivindica autoria sobre os arquivos aqui disponibilizados. Todos os créditos da tradução pertencem à Central de Tradução.

Caso a equipe responsável pelo projeto solicite a remoção do conteúdo, o repositório será atualizado conforme necessário.
