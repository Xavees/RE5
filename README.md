# Resident Evil 5 - Tradução PT-BR (Central de Tradução)

## Sobre

Este repositório tem como objetivo disponibilizar os arquivos da tradução em português brasileiro de **Resident Evil 5**, originalmente desenvolvida pela **Central de Tradução**.

Todo o crédito pelo trabalho de tradução, adaptação e distribuição pertence à equipe da Central de Tradução. Este repositório serve apenas como uma forma de armazenar e organizar os arquivos para uso pessoal e da comunidade.

## Créditos

**Tradução:** Central de Tradução

Agradecimentos a toda a equipe envolvida no projeto por tornar Resident Evil 5 mais acessível para jogadores brasileiros.

## Recomendação de Uso

A versão com **legendas traduzidas** é a opção mais recomendada para a melhor experiência possível, pois mantém a qualidade original das vozes do jogo enquanto fornece uma tradução de alta qualidade dos textos.

A versão **dublada por Inteligência Artificial** também está disponível, porém deve ser considerada uma alternativa experimental. A qualidade das vozes, entonações e interpretações pode variar significativamente em comparação com uma dublagem profissional, podendo apresentar falhas ou momentos menos naturais durante a jogatina.

## Aviso

Este repositório não reivindica autoria sobre a tradução. Caso solicitado pelos detentores dos direitos ou pela equipe responsável pelo projeto original, o conteúdo poderá ser removido.
